## 🚀 如何使用演示数据

### 方法 1：Supabase 控制台（推荐新手）

1. 登录 [Supabase Dashboard](https://supabase.com/dashboard)
2. 选择你的项目
3. 点击 **SQL Editor**
4. 新建查询，复制 `server/demo-data.sql` 的内容
5. 点击 **Run**
6. 等待完成，底部会显示 "演示数据初始化完成！"

### 方法 2：命令行（快速）

```bash
cd server
node scripts/init-demo-data.js
```

### 方法 3：直接 SQL（高级用户）

```bash
psql -h <your-db-host> -U <your-db-user> -d <your-db-name> -f server/demo-data.sql
```
