## 0. 安装应用（iOS）

**推荐：TestFlight**

1. 在 App Store 安装 **TestFlight**。  
2. 使用组织方发到你 **Apple ID 邮箱** 的测试邀请，或对方提供的 **TestFlight 公开链接**，按提示接受测试。  
3. 打开 **TestFlight**，找到 **Voxora**（或界面显示的应用名），点 **安装** / **更新**。  

换机时一般仍用同一 Apple ID 在 **TestFlight** 里安装即可，**通常不需要**为每台新 iPhone 单独再打一份「内测安装包」。若收不到邀请，请联系维护者在 App Store Connect 中核对你的邮箱或重新发送。

**备选（仅当团队明确使用 EAS 内部分发时）：**需先在真机上完成 **设备注册（UDID）**，再打开维护者提供的 **Expo 单次构建安装页**；新设备或新构建常需重新出包。详见 [iOS 分发说明（TestFlight 与 internal）](./ios-eas-joint-testing.md)。

## 1. 产品是什么

Voxora 是一款面向家庭的 **回忆记录与情感互动** 应用：你可以用图文记录「时光长廊」里的珍贵瞬间，也可以在 **「遇见未来」** 里和家人一起脑暴即将到来的聚会、旅行或纪念日，用文字或语音留言，最后由小结功能整理成可执行的方案建议。

数据与能力会随你的使用环境变化（例如是否连接自家后端、是否开启云端能力），下文会标注 **本机 / 局域网** 与 **需要网络** 的差异。

## 2. 界面总览（底部导航）

应用底部一般有五个区域（中间为「添加」按钮，具体以你安装的界面为准）：

| 区域 | 典型用途 |
|------|----------|
| **首页** | 家庭动态岛、情绪记录、**时光长廊**与**遇见未来**双卡片、回忆列表等 |
| **家庭** | 家庭空间相关能力（成员、拓扑、提醒等，以实际版本为准） |
| **添加（中间按钮）** | 快捷添加回忆、Vlog、打开语音指令等（点按或长按可能触发不同菜单） |
| **瞬间** | 与图片/瞬间流相关的浏览入口 |
| **我的** | 个人资料、设置等 |

语言可在系统或应用支持范围内切换；部分文案会随语言变化。

## 3. 首页：时光长廊

**时光长廊**展示已收录的回忆卡片（封面、标题、日期等）。你可以：

- **点按**某条回忆进入 **回忆详情**（若当前版本已接入该流程）。
- **向上滑动列表**浏览更多条目；列表底部会为底部导航留出空白，避免遮挡。
- 在「时光长廊 / 遇见未来」切换卡片上，可看到一句简短氛围文案（例如中文 **「翻翻就醉人」**），用于表达「轻轻翻阅回忆」的感觉。

---

## 4. 首页：遇见未来

「遇见未来」是家庭 **计划与讨论白板**：把「想一起做的事」收成一张张 **未来卡片**，家人可以各自留言，最后再 **小结**。

### 4.1 顶部的渐变区域

- 展示 **随机一句** 与家庭计划相关的轻松文案（**不再重复显示「遇见未来」标题**，避免与下方 Tab 文案重复）。
- 下方有小标签示意：讨论、清单分工、互相打趣等氛围。

### 4.2 新建未来卡片

1. 点 **「再画一笔未来」** 或空状态下的 **点亮第一件事** 等按钮。  
2. 在弹窗中填写：  
   - **想做的事**（标题，必填）  
   - **大概什么时候**（可选，自由文字即可，如「五一」「下周末」）  
   - **类型**：旅行 / 生日 / 派对 / 聚会  
3. 确认保存后，新卡片会出现在列表顶部。

### 4.3 卡片上你能看到什么

- **标题与日期**（若填写了日期）。  
- **状态**：「脑暴中」或「已定稿」（点锁图标可切换；用于表达讨论阶段，可按家庭习惯使用）。  
- **类型**（如「旅行」）显示在 **卡片右下方**。  
- **操作提示**：例如点按进入讨论、长按可删除卡片（以界面文案为准）。

### 4.4 进入某张卡片（讨论页）

1. **点按**卡片进入 **遇见未来 · 详情**。  
2. 顶部显示 **该卡片的标题**、类型与日期；可用 **铅笔** 修改标题（保存 / 取消）。  
3. **七嘴八舌**：  
   - 选择 **以谁的名义发言**（如：我、奶奶、爸爸、妈妈）。  
   - **文字**：在底部输入框书写后发送。  
   - **语音**：点麦克风，说完后转写成文字并作为一条留言出现（会带「语音」标记）。  
   - **待办**：每条留言旁可点 **增加到待办事项**（需服务端已配置 **Gemini**）；AI 会从该条发言提炼一条可执行待办并加入列表。点待办左侧方框可标记 **完成**。  
4. **小结一下**：  
   - 需要 **至少有一条留言** 再小结（否则会有提示）。  
   - 小结会汇总当前卡片上的发言；若服务端已配置 **Gemini**，会优先用 AI 生成中文结构化小结；否则会使用 **本地模板** 生成，仍可读、可分享。  
   - 小结会保存在本卡片内，下次打开仍可见。

### 4.5 删除卡片

在首页列表对卡片 **长按**，通过删除条确认删除（具体交互以当前版本为准）。

## 5. 添加回忆与其他入口

- **中间「添加」按钮**：通常可新建回忆、打开 Vlog、或使用 **语音指令**（说出类似「我要增加一个记忆」「打开瞬间」等，实际支持说法以应用内提示为准）。  
- **语音指令**可能依赖麦克风权限；识别结果会用于跳转或匹配回忆。

---

## 6. 权限与隐私提示

使用下列能力时，系统可能弹出权限请求（文案以系统与应用为准）：

- **相册 / 相机**：上传或拍摄回忆配图、Vlog 等。  
- **麦克风**：语音留言、语音指令、视频收音等。  
- **位置**（若功能启用）：与地点相关的记录或服务。

**遇见未来** 的卡片与留言会 **优先保存在本机**，并在能连上你配置的 **Express 后端** 时 **自动同步到服务器**（`GET/PUT /api/v1/future-plans`，数据落在服务端的 `data/future-plans/` 目录）。离线时仍可编辑，联网后会再次尝试上传。多设备之间以 **服务器上最后一次保存的列表为准**（简单合并策略）。若未部署后端或地址配置错误，则仅本机可见；换机前请确保已成功同步或自行导出备份。

---

## 7. 网络与「连上自己家后端」

- **小结、回忆列表、上传等** 若依赖你部署的 **Express 后端**，需要手机与服务器网络互通（同一 Wi‑Fi、内网穿透或公网地址等）。  
- 开发或部署脚本里常会配置 **`EXPO_PUBLIC_BACKEND_BASE_URL`**（或等价变量），指向你的 API 根地址；若配置错误，可能出现加载失败、小结失败等提示。  
- 遇到问题时，可先检查 **网络**、**后端是否启动**、以及应用内是否指向正确环境。

## 8. 常见问题（FAQ）

**Q：小结一直失败？**  
A：检查网络与后端地址；若未配置 AI，仍应返回本地模板小结。若完全失败，请看服务端日志或联系维护者。

**Q：遇见未来的标题写错了？**  
A：进入该卡片详情，点右上角 **铅笔** 修改标题并保存。

**Q：英文界面下时光长廊下的小字不一样？**  
A：部分文案按语言单独翻译，属于正常现象。
